# History / Superseded

This folder is the archive for BillBuffer documents that are **retired or
superseded**. It exists so the project's history is preserved as the docs evolve.

## Rules

- **Nothing is deleted.** When a document is replaced, it moves here (via `git mv`,
  preserving history) rather than being removed.
- **Leave a pointer.** The replacing document should carry a short note explaining
  what superseded the archived one and why.
- **Archived docs are not canonical.** Treat anything in this folder as historical
  context only — never build from it. The canonical, active docs live in the other
  `docs/` tiers.

## Contents

_Empty._ As of this reorganization, every BillBuffer document is still active and
lives in its tier under [`docs/`](../). Docs that were partially reconciled to the
locked architecture (e.g. `product/BILLBUFFER_MVP_DEFINITION.md`,
`product/BILLBUFFER_OPEN_QUESTIONS.md`) remain **active** — they carry in-place
reconciliation banners and were intentionally **not** archived.
